Installation:
  Installing may be a bit hard but download this entire repo as a folder on your computer.

  Setting up python:
    Install virtualenv package for python
    Use it to generate a new virtualenv with the name venv in the main directory. Can be done with "python -m virtualenv ./venv"
    Install all python packages needed to run the app. Just run the app and install whichever packages the error that pops up indicate. (Use google)

  Setting up flutter:
    In the main directory, run the command "flutter create ./flutter_app"
